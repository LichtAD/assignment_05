Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

How do you select an element from the DOM?

To select an element from the DOM, we can use getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector() and querySelectorAll() etc.

What is event delegation in the context of DOM and why is it useful?

Event delegation allows you to attach a single event listener to a parent element and handle events on its child elements through event bubbling. This is useful for managing many dynamic elements efficiently and reducing memory usage.

How do you manipulate an events attribute and styles using the DOM?

To manipulate events, use addEventListener() or assign event handlers directly (e.g., element.onclick = function() {}). For styles, modify the style property (e.g., element.style.color = 'blue';).
